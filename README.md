# Momentum (FitFlow)

<div align="center">
掌控身体的律动，记录纯粹的势能。  
_Control your rhythm, record your pure momentum._
</div>

---

## 📖 哲学 | Philosophy

> “这不仅仅是关于数字的记录，更是关于势能的积累。”  
> _"This is not just about tracking numbers—it's about accumulating kinetic potential."_

Momentum 剔除了一切花哨的社交功能和复杂的广告干扰。我们相信，最好的训练 App 应该是隐形的——在你需要时精准记录，在你专注时默默退后。这是属于你和钢铁之间的纯粹对话。

---

## ✨ 核心特性 | Features

- 🧘‍♂️ **极致极简 UI**  
  无边框、暗色沉浸模式，专为健身房昏暗环境与高度专注的训练者设计。
- 📊 **精准追踪**  
  无缝记录每一组的重量、次数与 RPE（主观疲劳感觉）。
- ⏱️ **智能休息计时**  
  内置悬浮与全局组间休息计时器（Rest Timer），结束时提供优雅提示音。
- 📚 **自定义动作库**  
  从分化训练到全身循环，自由创建训练计划与专属动作库。
- 🔒 **本地优先**  
  无强制联网要求，你的数据完全掌握在自己手中。

## 🛠️ 开发者指南 | Getting Started

Momentum 使用 Flutter 构建，支持多平台编译和开发。

#### 前置要求

- Flutter SDK（建议 3.x 及以上）
- Dart SDK

#### 本地运行

1. 克隆仓库

   ```bash
   git clone https://github.com/your-username/momentum.git
   cd momentum
   ```

2. 获取依赖

   ```bash
   flutter pub get
   ```

3. 运行应用  
   连接真机或启动模拟器后运行：

   ```bash
   flutter run
   ```

#### 核心目录结构

- `lib/models/`  
  数据模型（如 Workout, Exercise 等）
- `lib/pages/`  
  核心页面视图（首页、训练页、计划页等）
- `lib/widgets/`  
   UI 组件（如 exercise_card.dart, rest_timer_panel.dart）
- `lib/services/`  
  核心业务逻辑与本地化服务
