Momentum 摒弃了花哨的社交功能和复杂的广告。我们相信，最好的训练App应该是隐形的——在你需要时出现，在你要专注时退后。这是属于你和钢铁之间的对话。

训练记录

记录组数、重量与次数，勾选完成状态，一键新增组数，专注于当下训练流程。

休息计时

内置休息计时器与提醒，倒计时结束自动提示，节奏清晰不被打断。

自由定制

创建属于你的训练计划，从分化训练到全身循环。灵活的休息计时器与自定义动作库。
