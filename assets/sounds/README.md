# 音频资源说明

此文件夹用于存放组间休息倒计时结束的提醒音。

## 添加音频文件

请在此文件夹中放置一个名为 `alarm.mp3` 的音频文件作为提醒音。

建议使用：
- 短促的提示音（3-5秒）
- 循环播放时不会太刺耳的音频
- 格式：MP3 或 WAV

## 获取免费提示音

你可以从以下网站获取免费的提示音：
- https://pixabay.com/sound-effects/
- https://freesound.org/

或者使用在线工具生成简单的提示音。

## 注意

如果没有提供音频文件，应用会尝试使用系统默认提示音。
